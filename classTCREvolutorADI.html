Detailed Description

Class for the solution of the transport equation, derived from TCREvolutorBasis.

Author
Luca Maccione luca..nosp@m.macc.nosp@m.ione@.nosp@m.desy.nosp@m..de This class implements a second order operator splitting alternate-direction implicit (ADI) Cranck-Nicholson method. It essentially reimplements the method TCREvolutor::Run(...).
