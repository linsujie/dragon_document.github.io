Detailed Description

DRAGON main class.

Author
Luca Maccione luca..nosp@m.macc.nosp@m.ione@.nosp@m.desy.nosp@m..de This class takes care of reading user input and settings, creating the necessary galactic structure, the algorithm to solve the diffusion equation, the structure to contain the propagated nuclear densities and the output.
