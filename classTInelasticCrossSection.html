Detailed Description

Class containing a description of the total inelastic cross section, used to compute the disappearance rate of CRs.

Author
Luca Maccione luca..nosp@m.macc.nosp@m.ione@.nosp@m.desy.nosp@m..de Two models can be used up to now. One is the Galprop model for the cross section. The second model uses Webber (2003) cross sections.
