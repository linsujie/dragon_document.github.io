Detailed Description

Class for the solution of the transport equation, derived from TCREvolutorBasis.

Author
Luca Maccione luca..nosp@m.macc.nosp@m.ione@.nosp@m.desy.nosp@m..de This class implements a second order operator splitting implicit Cranck-Nicholson method. In order to speed up the solution, and also to compare with Galprop, we start with a very large time step, and after some iterations we reduce the time step. Minimum time step has to be of order of 100 yr to have reasonable solutions for nuclei, and of order of 1 yr for leptons.
