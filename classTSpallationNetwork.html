Detailed Description

Class containing the description of the network of spallation used to compute secondary CR source terms.

Author
Luca Maccione luca..nosp@m.macc.nosp@m.ione@.nosp@m.desy.nosp@m..de Two models can be used up to now. One is the Galprop model for the spallation cross section. The second model uses Webber (2003) cross sections. For electron and positron production also Kamae (2007) model can be used.
